# MeCard
A card about myself and what i do.

## Requirements to run

- Node.js
- git

### How to run
Inside a directory of your choice open the commandline and type

```
git clone https://github.com/MarkVN2/MeCard
cd MeCard
npm install
npm run dev
```
After that open **127.0.0.1:3000** in the web browser.
